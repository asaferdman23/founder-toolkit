# Privacy Policy

## Founder Toolkit Plugin

**Last updated:** March 31, 2026

### Overview

The Founder Toolkit plugin respects your privacy. We don't collect, store, or sell your personal or business data.

---

## Data Collection

### What We Don't Do
- ❌ We do NOT collect any personal information
- ❌ We do NOT store conversations on our servers
- ❌ We do NOT track your usage
- ❌ We do NOT share your data with third parties
- ❌ We do NOT require authentication or accounts

### What You Should Know
- All conversations happen **within your Claude Code session**
- Your business data (financial metrics, customer information, etc.) stays **entirely on your machine**
- Skills may save data to your **local memory files** (`~/.claude/projects/*/memory/`) — this data never leaves your computer
- You control what's in memory — you can edit or delete anytime

---

## External Services

### Claude API
- The plugin runs on Claude's API (same privacy as using Claude directly)
- See Claude's privacy policy: https://claude.ai/privacy

### GitHub
- The plugin code is hosted on GitHub (public repository)
- GitHub's privacy policy applies: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

### Analytics (Landing Page Only)
- Our landing page uses **Plausible Analytics** (privacy-first, no cookies)
- Plausible does NOT track individuals or use personal data
- See Plausible's privacy: https://plausible.io/privacy

### Web Search (Optional)
- If you use `/cofounder-market` or `/investor` skills with search enabled
- Those queries go through Claude's web search (not stored by our plugin)
- See Claude's privacy policy for search data handling

---

## Your Responsibilities

When using Founder Toolkit, **you are responsible for:**
- Keeping sensitive business information confidential
- Not sharing API keys, passwords, or credentials in conversations
- Complying with your organization's data retention policies
- Understanding that saved memory files are stored locally

---

## Data You Control

### Memory Files
Skills may save business context to local files for future sessions:
- Location: `~/.claude/projects/*/memory/`
- Contents: Business metrics, assumptions, customer data (whatever you input)
- You can: Edit, delete, or disable anytime
- We cannot: Access or see your memory files

### Example Memory Files
- `founder_metrics.md` — Your unit economics and financial data
- `market_analysis.md` — Competitive intelligence and TAM
- `pitch_records.md` — Past pitch prep work

**You own all memory files. Delete them anytime.**
